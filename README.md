# GetPhoneInfo
RetrofitDemo，调用聚合数据接口查询手机号归属地信息</br>
涉及技术:</br>
1、RxJava+Retrofit,GET方法获取网络数据（聚合数据）</br>
2、GSON解析json数据</br>
